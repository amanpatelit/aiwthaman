# Website Logo Files

## Logo Variants

### 1. **logo.svg** (200x200px)
- Full detailed logo with animations
- Features:
  - Cloud symbol (representing Cloud Computing)
  - AI neural network (representing AI/ML)
  - Letter "A" integrated design
  - Gradient colors: Blue (#0066ff) to Teal (#00ff99)
  - Glow effects and animated particles
  - Tech corner brackets
- Use: Main logo for website header, social media, presentations

### 2. **logo-simple.svg** (64x64px)
- Simplified version for favicon and small displays
- Features:
  - Minimalist cloud + AI design
  - Letter "A" element
  - Same gradient colors
  - Optimized for small sizes
- Use: Browser favicon, mobile icons, small UI elements

## Design Elements

### Color Palette
- Primary Blue: `#0066ff`
- Secondary Teal: `#00ff99`
- Accent Blue: `#00a8ff`

### Symbolism
- **Cloud**: Represents cloud computing expertise (AWS, Azure, GCP)
- **Neural Network**: Represents AI and machine learning capabilities
- **Letter A**: Personal branding for "aiwithaman"
- **Tech Brackets**: Modern, technical aesthetic

## Usage Guidelines

1. **Minimum Size**: 32x32px for logo-simple.svg
2. **Background**: Works best on dark backgrounds
3. **Spacing**: Maintain 10px padding around logo
4. **Colors**: Do not modify gradient colors
5. **Format**: SVG preferred for scalability

## File Locations
- `/logo.svg` - Main detailed logo
- `/logo-simple.svg` - Simplified favicon version
- Used across all HTML pages as favicon

## Integration
All website pages now use the new logo:
- index.html
- about.html
- skills.html
- projects.html
- contact.html
- 404.html

The logo is also integrated into the navigation bar with the existing animated SVG design.
